#### Instructions for blog.zip in Cloud9

<pre>
1. Upload zip
2. Navigate to the folder in terminal
3. rails server -b $IP -p $PORT
4. Preview button --> Preview Running Application
5. Pop Out Into New Window (button by the word "Browser")
